Questão 1 — Processamento de lista com for: filtragem de pares e ímpares

Considere a lista abaixo:

List<int> valores = [12, 7, 3, 18, 10, 25, 2, 9, 30];

Usando apenas for, faça:

  • Crie uma nova lista contendo apenas os valores pares.
  • Crie outra lista contendo apenas os valores ímpares.
  • Mostre as duas listas separadamente.
  • Mostre a quantidade de pares e a quantidade de ímpares.
Questão 2 — Construção de lista com while: cadastro até "fim"

Crie um programa que leia nomes do usuário e os armazene em uma lista usando um while.

Regra: o usuário pode digitar quantos nomes quiser, até digitar "fim".

  • Mostre quantos nomes foram cadastrados.
  • Mostre a lista completa.
  • Mostre o primeiro e o último nome cadastrados.
  • Remova o primeiro nome usando removeAt(0) e mostre a lista atualizada.

Dica: use stdin.readLineSync() e compare strings.

Questão 3 — Inserção e remoção em posições específicas

Considere a lista abaixo:

List<int> numeros = [4, 8, 12, 16, 20, 24];

Faça o que se pede usando apenas comandos já vistos em aula (for, while, insert, removeAt, indexOf):

  • Peça ao usuário um número e uma posição.
  • Insira o número exatamente na posição informada usando insert().
  • Em seguida, peça ao usuário outra posição.
  • Remova o valor que está nessa posição usando removeAt().
  • Por fim, mostre todos os elementos da lista usando um for.

Atenção: valide se as posições informadas são válidas antes de inserir ou remover.

Questão 4 — Classes e Objetos: sistema simples de alunos

Com base no exemplo da classe Empregado mostrado em sala, crie uma classe chamada Aluno contendo:

  • String? nome;
  • int? idade;
  • double? nota;

A classe deve possuir os métodos:

  • void aumentaNota(double x) – soma x à nota.
  • void diminuiNota(double x) – subtrai x da nota.
  • void mostraDados() – imprime nome, idade e nota.

No main():

  • Crie 3 objetos Aluno para armazenar alunos, 1, 2, 3.
  • Mostre os dados de todos os alunos usando o mostraDados para cada objeto
  • Aumente a nota do primeiro aluno e diminua a nota do segundo.
  • Mostre novamente todos os dados.
