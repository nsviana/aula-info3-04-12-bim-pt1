Questão 1 — Processamento de lista com for: filtragem de pares e ímpares

Considere a lista abaixo:

List<int> valores = [5.5, 7.5, 8.7, 9.1, 4.3];

Usando a estrutura for, faça:

  • Crie uma nova lista contendo apenas os valores maiores que 7.0.
  • Crie outra lista contendo apenas os valores menores que 5.0.
  • Mostre as duas listas separadamente.
  • Mostre o tamanho de cada lista
Questão 2 — Construção de lista com while: cadastro até "fim"

Crie um programa que leia um número real do usuário e os armazene em uma lista usando um while.

Regra: o usuário pode digitar quantos números quiser, até digitar 0.0.

  • Mostre quantos números foram cadastrados.
  • Mostre a lista completa.
  • Mostre o primeiro e o último número cadastrados.

Dica: use stdin.readLineSync() e compare números.

Questão 3 — Inserção e remoção em posições específicas

Considere a lista abaixo:

List<int> nomes = ["Pedro", "Ana", "José", "Maria", "Aline"];

Faça o que se pede usando apenas comandos já vistos em aula (for, while, insert, removeAt, indexOf):

  • Peça ao usuário uma posição.
  • Insira um nome 'Alan' exatamente na posição informada usando insert().
  • Em seguida, peça ao usuário outra posição.
  • Remova o valor que está nessa posição usando removeAt().
  • Por fim, mostre todos os elementos da lista usando um for.
Questão 4 — Classes e Objetos: sistema simples de alunos

Com base no exemplo da classe Empregado mostrado em sala, crie uma classe chamada Produto contendo:

  • String? nome;
  • int? quantidade;
  • double? preco;

A classe deve possuir os métodos:

  • void aumentaPreco(double x) – soma x ao preco.
  • void diminuiPreco(double x) – subtrai x ao preco.
  • void mostraDados() – imprime nome, quantidade e preco.

No main():

  • Crie 3 objetos Produto para armazenar produtos, 1, 2, 3.
  • Mostre os dados de todos os produtos usando o mostraDados para cada objeto
  • Aumente o preco do primeiro produto e diminua o do segundo.
  • Mostre novamente todos os dados.
